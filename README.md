# Emotion Detection using Deep Learning (CNN)

This project implements an intelligent emotion detection system using Convolutional Neural Networks (CNN) and the FER-2013 dataset.

## Features
- Facial emotion recognition
- CNN architecture
- Model training and evaluation
- F1-score and confusion matrix
- SHAP explainability
- External image prediction

## Technologies
- Python
- TensorFlow / Keras
- NumPy
- Pandas
- Matplotlib
- OpenCV
- SHAP

## Dataset
FER-2013 facial emotion dataset.

## Results
The model was trained to recognize:
- Angry
- Happy
- Sad
- Fear
- Neutral
- Surprise
- Disgust
